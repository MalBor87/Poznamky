Virtualizácia

Skontrolujte podporu virtualizácie na Ubuntu

egrep -c '(vmx|svm)' /proc/cpuinfo

Ak príkaz vráti hodnotu 0, váš procesor nedokáže spustiť KVM. Teraz skontrolujte, či váš systém môže používať akceleráciu KVM.

sudo kvm-ok

Ak kvm-ok vráti chybu, že nie je možné použiť akceleráciu KVM, je potrebne vykonať inštaláciu cpu-checker.

sudo apt install cpu-checker

reštartujte server Následne aktualizujte repozitáre

sudo apt update

Potom nainštalujte základné balíky KVM pomocou nasledujúceho príkazu

sudo apt install qemu-kvm libvirt-daemon-system libvirt-clients bridge-utils
